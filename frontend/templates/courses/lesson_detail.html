Introduction to Django Framework

15 min • Beginner Level

What is Django?

Django is a high-level Python web framework that encourages rapid development and clean, pragmatic design. Built by experienced developers, it takes care of much of the hassle of web development, so you can focus on writing your app without needing to reinvent the wheel.

Key Features of Django

  • Fast Development: Django was designed to help developers take applications from concept to completion as quickly as possible.
  • Secure: Django takes security seriously and helps developers avoid many common security mistakes.
  • Scalable: Some of the busiest sites on the web leverage Django's ability to quickly scale.
  • Versatile: Django can be used to build almost any type of website — from content management systems to social networks.

Django MVT Architecture

Django follows the MVT (Model-View-Template) pattern, which is similar to MVC but with some key differences:

Model

Defines the structure of your database and handles data logic

View

Contains the business logic and interacts with models

Template

Handles the presentation layer (HTML)

Setting Up Django

Let's get Django installed and create your first project:

Step 1: Install Django

bash 
# Create a virtual environment
python -m venv venv

# Activate virtual environment
# On Windows:
venv\Scripts\activate
# On macOS/Linux:
source venv/bin/activate

# Install Django
pip install django

Step 2: Create a Django Project

bash 
# Create a new Django project
django-admin startproject myproject

# Navigate to project directory
cd myproject

# Run development server
python manage.py runserver

Step 3: Understanding Project Structure

After creating a project, you'll see this structure:

text 
myproject/
├── manage.py           # Command-line utility
└── myproject/
    ├── __init__.py    # Python package marker
    ├── settings.py    # Project settings
    ├── urls.py        # URL declarations
    ├── asgi.py        # ASGI configuration
    └── wsgi.py        # WSGI configuration

Creating Your First App

Django projects are organized into apps. Let's create one:

bash 
python manage.py startapp blog
Congratulations! You've just created your first Django project and app. In the next lesson, we'll dive into models and database design.
